# TreeChat - 愿景与功能设计方案

## 1. 项目简介

TreeChat 旨在打破传统 AI 对话的"线性锁定"，将聊天室从传统的"单一链表"模式进化为"多分支树状结构"。用户可以在对话的任何节点开启平行分支，自由探索不同的逻辑可能性，并最终将多个分支的智慧合并为一个结论。

## 2. 核心痛点

- **上下文污染**：后续对话常受前面无关信息干扰。
- **思维受限**：无法在同一上下文中并行测试不同方向。
- **难以收敛**：散乱的对话难以汇总成结构化的成果。

## 3. 核心功能

### 3.1 分支探索 (Forking) ✅ 已实现

- **任意点分叉**：用户可以点击对话历史中的任意一条消息，以此为起点开启新的对话分支。
- **平行对比**：在不同的分支中可以设置不同的 System Prompt 或调用不同的 AI 模型。

### 3.2 节点管理 (Node Management) ✅ 部分实现

- **单节点删除**：✅ 支持删除单个节点（不级联删除子节点，子节点保留为独立浮动节点）。
- **批量删除**：✅ 多选节点后一键批量删除，附带确认弹窗保护。
- **自由拖拽与缩放**：✅ 每个节点支持拖拽定位和右下角手柄拉伸缩放。
- **剪枝 (Pruning)**：🔜 计划中 — 级联删除目标节点及其所有子孙节点。
- **折叠与展开**：🔜 计划中 — 收起子树保持工作区整洁。

### 3.3 节点合并 (Merging) ✅ 已实现

- **多选合并**：允许用户选择多个非连续或不同分支的节点。
- **AI 智能综述**：AI 会自动提取所选节点的核心内容，进行逻辑压缩与融合，生成一份综合上下文摘要作为新分支的起始节点。
- **合并配置**：合并时可指定使用的 API 配置和模型。

### 3.4 交互设计 ✅ 已实现

- **无限画布**：基于 React Flow 引擎的缩放与拖拽体验，支持 0.1x - 4x 缩放范围。
- **视觉导航**：MiniMap 缩略图，不同角色的节点使用不同颜色标识（User=蓝, System=灰, Assistant=翡翠绿）。
- **全局安全防护**：全局 Toaster 浮动消息提示 + Confirm Dialog 危险操作拦截。
- **Markdown/纯文本双模式**：每个节点可一键切换 Markdown 富文本渲染与纯文本编辑模式。
- **一键复制**：快速复制节点全量内容到剪贴板。
- **图片预览**：支持点击缩略图全屏预览上传的图片。

### 3.5 数据主权与模型解耦 ✅ 已实现

- **一切以数据为源**：对话树生成的所有内容（节点流、状态、提示词）均为开放的 JSON 数据格式，支持全量的**导出与导入**。导出文件自动以当前 Tab 名称命名，方便管理。
- **自由切换各类 AI 端点**：底层完全解耦特定大模型厂商的 API 锁定。支持 Google Gemini（原生 API）和所有 OpenAI Compatible 端点（OpenAI、DeepSeek、Claude、Ollama 等）。用户可以在每个节点处随时切换不同的 AI 端点，方便在平行分支下横向对比验证不同模型的回答能力。
- **远程模型列表获取**：API 配置面板支持一键从远程端点拉取可用模型列表，无需手动输入模型名称。

### 3.6 多模态支持 ✅ 已实现

- **图文混合输入**：User 节点支持上传多张图片，底层自动进行 Base64 转码并打包为多模态消息投喂给 AI。
- **图片管理**：支持单张图片删除、点击放大预览。

### 3.7 数据持久化 ✅ 已实现

- **IndexedDB 存储**：所有数据（节点、边、Tab、API 配置）持久化到浏览器 IndexedDB，容量不受 LocalStorage 5MB 限制。
- **智能防抖写入**：300ms 防抖减少频繁写盘开销。
- **安全刷入机制**：`beforeunload` 事件触发时立即刷入待写数据，减少页面关闭时的数据丢失。

## 4. 技术栈 (已落地)

| 层级         | 技术                        | 说明                                            |
| ------------ | --------------------------- | ----------------------------------------------- |
| **核心框架** | Next.js 16 (App Router)     | 服务端渲染 + API 路由                           |
| **UI 引擎**  | @xyflow/react v12           | React Flow 节点画布                             |
| **状态管理** | Zustand 5 + idb-keyval      | 单 Store + IndexedDB 持久化                     |
| **AI 接入**  | Vercel AI SDK 6             | `@ai-sdk/google` + `@ai-sdk/openai` 双 Provider |
| **样式**     | Tailwind CSS v4             | `@tailwindcss/typography` 排版插件              |
| **富文本**   | react-markdown + remark-gfm | GFM 语法支持                                    |
| **字体**     | Geist Sans + Geist Mono     | Google Fonts 托管                               |
| **图标**     | Lucide React                | 轻量矢量图标库                                  |
